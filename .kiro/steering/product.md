# NeuroTwin Product Overview

NeuroTwin is a cognitive digital twin platform that creates a living AI replica of a user's mind.

## Core Purpose
- Learn user's language style, decision patterns, and productivity rhythms
- Act autonomously across connected platforms (chat, email, calendar, calls)
- Handle communications and tasks while mimicking the user's personality

## Key Features
- **Cognitive Signature Model (CSM)**: Stores personality, tone, habits, decision patterns
- **Vector Memory Engine**: Long-term memory using embeddings
- **Automation Brain**: Executes workflows across connected apps
- **Voice Identity Engine**: Handles real phone calls using cloned voice
- **Behavioral Safety Layer**: Kill-switch, permissions, audit logs

## Cognitive Blend Slider
Users control how much "human" vs "AI logic" their Twin uses (0-100% scale).

## Integrations
WhatsApp, Telegram, Slack, Gmail, Outlook, Google Meet, Zoom, Google Docs, Microsoft Office, Calendar, CRM tools.

## Safety Principles
- Full audit logs required
- Permission-based actions
- Kill-switch for all automations
- No financial/legal actions without explicit approval
